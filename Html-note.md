# Html note

2016-12-19
```
<a href="/index.html">本文本</a> 是一个指向本网站中的一个页面的链接。</p>

使用 Target 属性，你可以定义被链接的文档在何处显示。
当使用命名锚（named anchors）时，我们可以创建直接跳至该命名锚（比如页面中某个小节）的链接，这样使用者就无需不停地滚动页面来寻找他们需要的信息了。
<map>	定义图像地图。
<area>	定义图像地图中的可点击区域。

每个表格由 table 标签开始。
每个表格行由 tr 标签开始。
每个表格数据由 td （table data）标签开始

no-breaking 空格由和号开始 ("&")，然后是字符"nbsp"，并以分号结尾(";")。
<ol>	定义有序列表。

块级元素在浏览器显示时，通常会以新行来开始（和结束）。
例子：<h1>, <p>, <ul>, <table>
内联元素在显示时通常不会以新行开始。
例子：<b>, <td>, <a>, <img>

HTML5 语义元素
header	定义文档或节的页眉
nav	定义导航链接的容器
section	定义文档中的节
article	定义独立的自包含文章
aside	定义内容之外的内容（比如侧栏）
footer	定义文档或节的页脚
details	定义额外的细节
summary	定义 details 元素的标题

scheme://host.domain:port/path/filename
解释：
scheme - 定义因特网服务的类型。最常见的类型是 http
host - 定义域主机（http 的默认主机是 www）
domain - 定义因特网域名，比如 w3school.com.cn
:port - 定义主机上的端口号（http 的默认端口号是 80）
path - 定义服务器上的路径（如果省略，则文档必须位于网站的根目录中）。
filename - 定义文档/资源的名称
URL 的英文全称是 Uniform Resource Locator，中文也译为“统一资源定位符”。

颜色相关：
http://www.w3school.com.cn/html/html_colors.asp

必需的属性：
请始终对图像使用 alt 属性。当图像无法显示时该属性很重要。
请始终定义图像尺寸。这样做会减少闪烁，因为浏览器会在图像加载之前为图像预留空间。
请尽量避免代码行超过 80 个字符。
```
